關於本站

從pixnet搬過來,為了給大家一個簡潔的網站(大概吧…)
從國一以來受到網路的幫助,所以也想為網路教學貢獻點心力
目前專寫網路(中文)上尚未出現相關教學或翻譯英文教學文為主(弄點獨家的,不然都沒人看)

歷史:

這個網站是2015/06/01才出現在這裡的,但在這之前文章是在sln98106.pixnet.net(現已關閉)搬過來的
這要從三年前(國一下學期)說起…
有一天突然被ocsb.pw(死鏈)的站長提起說要不要弄個教學部落格,那時我對部落格教學很感興趣(因為那時跟電腦有關係的知識都是從那學來的,所以也想像那些人一樣對網路有貢獻)
所以我就答應他架個部落格(其實這和他原本邀我建立部落格目的不一樣),因為當時能力(知識)有限,所以先選擇pixnet將就一下,2013/12/12,第一篇文章誕生了
後來因為是學生的因素,文章以「超緩慢」的速度更新,至2015/06/01前只有11篇文章(慚愧阿~
後來看pixnet越來越不慣(其實就是是廣告沒分紅,然後又拿不下來)所以有了自己架站的念頭
後來有了一點資金(其實是爸媽給的,感謝他們的偉大吧),買了塊raspberry pi 2<謎之聲:買錯東西啦你,是伺服器!伺服器!怎麼買到開發板去了
經過一個禮拜的調整和優化後,本站於2015/06/01正(ㄇㄛˋ)式(ㄇㄛˋ)上線,一個禮拜後將舊有部落格關閉

本站事件

2013/12/12 - sln98106.pixnet.net(本站前身)第一篇文章誕生
2015/06/01 - 本站正(ㄇㄛˋ)式(ㄇㄛˋ)上線,暫名為「墜落至IT的世界」
2015/06/02 - 本站第一篇文章發布
2015/06/08 - 關閉sln98106.pixnet.net
2015/08/29 - 本站正式採用https加密連線
2015/12/01 - 本站變更域名 sln98106.ddns.net -> bxstudio.cc
2016/01/01 - 本站正式更名為「某次元的教學blog」和變更佈景主題(外觀/樣板)至「evolve」以及再次變更域名 blog.bxstudio.cc
2016/08/10 - 第一個交換連結出現了!! 2016/12/13 - 謎之聲離開，全站轉換域名至blog.bennyling.cc
2022/08/06 - 伺服器更換，由raspberry pi 2B改為tinker Board 2s，感謝RPI 2這7年的付出

 - 你正在瀏覽我們的網站,別問我怎麼知道的

你知道嗎?

這個網站是用raspberry pi 2 + Nginx + Mysql + WordPress 架出來的,所有設定都是由Benny親自調整,只使用了一半的系統資源
謎之聲不是Benny腦補出來的,真的是一個存在的人,每次當Benny寫好草稿,都會先拿去給謎之聲吐槽一下再發表出去,即使是關於頁面也不例外,這是他的個人頁面(個人頁面也很神祕阿~)
如果你有看過本站前身,你會發現少了兩篇文章,因為Benny發現內容嚴重錯誤,撤下來了(所以成為了黑歷史)
你以為「本站事件」的最後一項「你正在瀏覽我們的網站,別問我怎麼知道的」是每天更新的?不,這用javascript就能解決了
「你知道嗎?」的內容是由謎之聲寫的,不然Benny不會寫自己的黑歷史
